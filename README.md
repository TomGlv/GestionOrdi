# GestionOrdi

## Description
GestionOrdi est un projet Java permettant de gérer des ordinateurs dans une organisation.  
Il inclut des fonctionnalités de suivi, d'ajout, de suppression et de modification des ordinateurs.

## Fonctionnalités
- Ajouter un ordinateur
- Supprimer un ordinateur
- Modifier les informations d'un ordinateur
- Afficher la liste des ordinateurs

## Technologies
- Java
- Eclipse
- Git / GitHub

## Installation
1. Cloner le dépôt :
git clone https://github.com/TomGlv/GestionOrdi.git
2. Ouvrir le projet avec Eclipse.
3. Compiler et exécuter `Main.java`.
